The World Wide Web - Tim Berners-Lee's Gift

Important Facts

  • The World Wide Web was created in 1989, with the first web browser being developed in 1990.
  • CERN would make the Web free to use for all in 1993, a decision that led to its rapid adoption.
  • Despite having only been available to the public for 26 years, the Web has transformed society and is the defining technology of the 21st Century.
  • The Web is designed such that a tired CS student in a 11am practical can develop a vaguely functional website.
